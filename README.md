# Multi-Modal AI Data Annotation System

## Overview
This project simulates a real-world AI data annotation pipeline used in machine learning training workflows. It includes text classification, image labeling simulation, data cleaning, and AI response evaluation.

---

## Modules

### 1. Text Classification
- Intent detection (Request / Complaint / Query / Feedback)
- Sentiment classification
- Category tagging

### 2. Image Labeling (Simulated)
Bounding box style annotation for object detection tasks.

### 3. AI Response Comparison
Evaluation of AI-generated outputs based on:
- Clarity
- Accuracy
- Completeness
- Readability

### 4. Data Cleaning
- Duplicate removal
- Spelling correction
- Normalization of text

- Data Annotation

## Purpose
To simulate real-world AI training workflows used in building machine learning datasets.
