# Annotation Guidelines

## Text Classification Rules
- REQUEST → user asks for action
- COMPLAINT → negative feedback or issue
- INFORMATION → factual statement

## Image Annotation Rules
- Identify visible objects
- Assign correct label
- Use bounding box format [x1, y1, x2, y2]

## Quality Rules
- Ensure consistency in labeling
- Remove ambiguity in classification
- Follow structured format strictly
